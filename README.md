# Pacman RL 

RL course project of tabular RL agent for simplified Pacman game.

## Structure
- `src/pacman_rldp/` — library code.
- `scripts/` — scripts for training, evaluation or manual play.
- `configs/` — yaml configs for the various parts.
- `results/` — local reults (not comited, except `results/important/`).
- `notebooks` — any .ipynb files (if needed).

## Quick start
```bash
python -m venv .venv
pip install -e .
```
